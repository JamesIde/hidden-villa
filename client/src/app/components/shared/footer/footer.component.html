HiddenVilla

A mobile-first hotel booking website where users can place bookings for rooms, make payments with Stripe, and login/register their details for a profile.

This project was created with the following: Angular, Node, Express, Prisma and PlanetScale. Other tech used includes MaterialUI, DaisyUI, TailwindCSS and Stripe. This is a fullstack TypeScript application.

This was my first big project with Angular, Prisma and PlanetScale (serverless SQL), which provided me the opportunity to nail in concepts about data fetching, user authentication with access, refresh tokens and cookies, and get the feel of developing a mobile-first application with the eventual goal of turning it into a PWA.
